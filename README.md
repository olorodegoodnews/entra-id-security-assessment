# Microsoft Entra ID Security Assessment

## Project Description

This project documents a security assessment of a Microsoft Entra ID tenant. The objective is to review identity management components, authentication methods, role assignments, and audit capabilities to understand how Microsoft Entra ID supports enterprise identity and access management (IAM).

The assessment follows a structured approach similar to how a cloud security or IAM analyst reviews an organization's identity infrastructure.

## Business Scenario

A fictional organization uses Microsoft Entra ID to manage identities for employees, devices, and cloud resources.

As a Cloud Security Analyst, my responsibility is to review the tenant configuration, examine identity-related resources, and document security observations and recommendations based on industry best practices.
## Objectives

- Review Microsoft Entra ID tenant information.
- Examine user identity management.
- Review group management.
- Analyze authentication methods.
- Review role assignments.
- Investigate audit and sign-in logs.
- Identify security best practices.

  
- Document findings professionally.
